Variable Scope

Variables live within pairs of curly brackets like these {}

Outside of those brackets, they cannot be accessed, so you can reuse variable names


			private void example()
			{
				int intNumA = 0;
				
				// We can access and read/modify intNumA here
				
				for (int i = 0; i < 10; i++)
				{
					// We can access intNumA here, because we are still within the original {}'s
					intNumA = intNumA + 2;
					i = intNumA * i;
				}
			}
			
			private void anotherMethod()
			{
				// We cannot see intNumA in here, because it is not within that block of {}'s
			}

How to declare/initialise an array

Declaring Arrays

To declare an array in C#, use the following syntax


			datatype[] arrayName;

where:

- datatype is used to specify the type of elements in the array

- arrayName specifies the name of the array

For example


			double[] balance;

Initialising Arrays

Declaring an array does not initialize the array in the memory. When the array variable is initialized, you can assign values to the array.

Array is a reference type, so you need to use the new keyword to create an instance of the array. For example,


			double[] balance = new double[10];

Assigning Values to Array Elements

You can assign values to individual array elements, by using the index number like -


			double[] balance = new double[10];
			balance[0] = 4500.0;

You can assign values to the array at the time of declaration, as shown −


			double[] balance = { 2340.0, 4523.69, 3421.0};

You can also create and initialise an array, as shown -


			int [] marks = new int[5]  { 99,  98, 92, 97, 95};

Accessing Array Elements

An element is accessed by indexing the array name. This is done by placing the index of the element within square brackets after the name of the array. For example,


			double salary = balance[9];

The following example, demonstrates the above-mentioned concepts of declaration, assignment, and accessing arrays −


			int[] intExampleArray = new int[10]; // intExampleArray is an array of 10 integers

            // initialize elements of array intExampleArray
            for (int i = 0; i < 10; i++)
            {
                intExampleArray[i] = i + 100;
            }

            // output each array element's value
            for (int j = 0; j < 10; j++)
            {
                Console.WriteLine("Element[{0}] = {1}", j, intExampleArray[j]);
            }
			

			Output:

Simple number validation

 

			bool booNumCheck = true;
            int intNum = -999; // error value
            while (booNumCheck == true)
            {
				int intMin = 1; // minimum acceptable value
				int intMax = 10; // maximum acceptable value
                Console.Write("Enter a number between " + intMin + " and " + intMax + ": ");
                
                while (!int.TryParse(Console.ReadLine(), out intNum))
                {
                    Console.Write("The value must be an integer, try again: ");
                }

                if (intNum >= intMin && intNum <= intMax)
                {
                    // number is between intMin and intMax inclusive
                    booNumCheck = false;
                }
                else
                {
                    // number is invalid
                    Console.WriteLine("Invalid input, must be between " + intMin + " and " + intMax);
                }
            }

            Console.WriteLine("intNum is: " + intNum);
			

			Output:
			
			Enter a number between 1 and 10: 11
			Invalid input, must be between 1 and 10
			Enter a number between 1 and 10: 2
			intNum is: 2
